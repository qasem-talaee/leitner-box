# Leitner Box
Leitner Box with python that you can add several boxes.
The software makes studying easy.Database is sqlite3.

## Installation
```bash
sudo apt install python3-venv python3-pip
```
```bash
python3 -m venv ./venv
```
```bash
source ./venv/bin/activate
```
```bash
pip install -r req.txt
```
```bash
python3 main.py
```
